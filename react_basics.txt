React is a javascript library for building user interfaces (UI).

React is used to build a single page applications 
It allows to builld reusable UI components.

It is a frontend JS library. 

Developed by Jordan Walke.

aka React.js or ReactJS.





HOW IT WORKS  :
-> reat creates a Virtual DOM memory.
React creates a virtual DOM in memory, where it does all the necessary manipulating, before making the changes in the browser DOM.
-> react only changes what needs to be changed.
